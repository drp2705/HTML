HTML BASICS

Question 1: Define HTML. What is the purpose of HTML in web development?

-> HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web.
   It defines the layout of web pages using elements like display text, images, links, forms, and other content.

Purpose of HTML in web development:
Provides the structure of web pages.
Organizes content into headings, paragraphs, lists, tables, etc.
Allows embedding of media like images, audio, and video.
Works together with CSS and JavaScript to build websites.

Question 3: What is the difference between block-level elements and inline elements in HTML? Provide examples of each.
->
	Block-level elements
	Start on a new line.
	Take up the full width available by default.
	Can contain other block-level or inline elements.

Examples:
<div>, <p>, <h1>–<h6>, <ul>, <ol>, <li>, <section>, <header>

	Inline elements
	Do not start on a new line.
	Usually contain only text or other inline elements.

Examples:
<span>, <a>, <img>, <strong>, <em>, <label>


HTML FORMS

Question 1: What are HTML forms used for? Describe the purpose of the input, textarea, select, and button elements.
->

HTML forms are used to collect information from users and send it to a server for processing. Forms are essential for actions.

Form elements:
<input>
Used for single-line user input. It has different types, such as:
text for plain text
password for hidden characters
email for email addresses

<textarea>
Used for multi-line text input. Ideal for comments, messages, descriptions, or long text entries.

<select>
Creates a drop-down list of options. It is used when the user needs to choose one (or sometimes multiple) items from a predefined list.

<button>
Used to create clickable buttons. The most common is type="submit" to send the form data.

Question 2: Explain the difference between the GET and POST methods.
->

GET method:
Sends form data as part of the URL (query string).
Data is visible in the browser’s address bar.
Not suitable for sensitive information.

POST method:
Sends data in the body of the request, not in the URL.
More secure and private.
Allows much larger amounts of data.

HTML TABLES
                                                                                                    
Question 1: Explain the structure of an HTML table and the purpose of each element
->

An HTML table is used to display data in rows and columns.
<table>
<tr> (Table Row)
Defines a row within the table. A table is made up of multiple <tr> elements.
<th> (Table Header Cell)
Defines a header cell within a row.
Text in <th> is bold by default and centered 
Understand the meaning of each column.
<td> (Table Data Cell)
Defines a standard cell containing data within a row.
<thead>
Groups and defines the header section of the table.


Example structure:

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
  </thead>
  <tr>
    <td>Dhvani</td>
    <td>25</td>
  </tr>
</table>

Question 2: Difference between colspan and rowspan with ex
Colspan:
<tr>
  <td colspan="3">This cell spans 3 columns</td>
</tr>

Rowspan:
<tr>
  <td rowspan="2">This cell spans 2 rows</td>
  <td>Row 1, Col 2</td>
</tr>
<tr>
  <td>Row 2, Col 2</td>
</tr>
